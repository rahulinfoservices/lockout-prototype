# Lockout Prototype

## Overview

This is a prototype for a lockout app. It is a React Native app that allows lockout admins to view their device health and security alerts.

## Getting Started

### Prerequisites

- Node.js
- PNPM

### Installation

1. Clone the repository
2. Install dependencies with `pnpm install`
3. Add the Firebase configuration files for iOS and Android. You can download it from the Firebase console.
4. Run the app with `pnpm start`
